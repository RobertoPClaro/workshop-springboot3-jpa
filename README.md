# Sobre o projeto

Eu criei uma API de estudos usando o Spring Boot, com o objetivo de aprender a desenvolver uma aplicação Java com as principais funcionalidades de uma API RESTful.
Durante o desenvolvimento da API, abordei diversos temas importantes, como a criação de um projeto Spring Boot Java, a implementação do modelo de domínio, a estruturação das camadas lógicas (resource, service, repository), a configuração do banco de dados de teste (H2), o povoamento do banco de dados, a implementação do CRUD (Create, Retrieve, Update, Delete) e o tratamento de exceções.

```bash

# executar o projeto
./mvnw spring-boot:run
```
